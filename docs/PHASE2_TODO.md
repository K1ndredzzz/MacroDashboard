### Phase 2A：情景模拟器 ⭐⭐⭐（核心特色功能）

#### 1. 利率冲击模拟器（预计 3 天）
**功能描述**：
- 输入：利率变化幅度（+/- 50-200 bps）
- 输出：对汇率、股市、债券的影响预测
- 基于历史相关性计算

**技术实现**：
```
后端：
- 新增 API 端点：POST /api/v1/simulation/shock
- 新增服务：backend/api/app/services/simulation.py
- 算法：影响 = 相关系数 × 冲击幅度 × 敏感度系数

前端：
- 新增组件：frontend/src/components/ShockSimulator.tsx
- 交互式输入界面（滑块 + 数值输入）
- 结果可视化（雷达图 + 柱状图）
```

**验收标准**：
- [ ] 支持利率冲击输入（50-200 bps）
- [ ] 计算对主要指标的影响
- [ ] 可视化影响结果
- [ ] 显示置信区间

## 技术难点

### 1. 冲击模拟的准确性
**挑战**：简单的线性模型可能不够准确
**解决方案**：
- 使用历史事件校准模型
- 提供置信区间
- 明确标注"模拟结果仅供参考"

### 2. 计算性能
**挑战**：互相关计算可能较慢
**解决方案**：
- Redis 缓存结果
- 限制最大计算窗口
- 异步计算 + 进度提示

### 3. 数据质量
**挑战**：历史数据可能不完整
**解决方案**：
- 数据质量检查
- 前端显示数据可用性
- 支持部分指标分析
